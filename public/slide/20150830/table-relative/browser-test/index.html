• table 要素の構造は全て同じで、position: relative を指定する要素が異なっています
    • 中央のセルの子要素として、赤色の「A」「B」「C」「D」を absolute 配置しています
    • table-caption として、灰色のボックスを置き、その中で黄色の「@」を absolute 配置しています
  • 行ごとのテストケース
    • Case-1-* は table-collapse: separate 指定
    • Case-2-* は table-collapse: collapse 指定
    • Case-3-* は table-collapse: collapse 指定し、background を付ける要素に background-clip: padding-box を指定
  • 列ごとのテストケース
    • Case-*-1 は table 関連要素には指定無し
    • Case-*-2 は table に relative 指定
    • Case-*-3 は tbody に relative 指定
    • Case-*-4 は tr に relative 指定
    • Case-*-5 は td に relative 指定
    • Case-*-6 は td に relative 指定し、td の background を外している
  • 上部と下部のテストケース
    • 上部（青ラベル Case-*-*）のテストケースは「子孫の基点」に注目したテストです
    • 下部（赤ラベル Case-*-*-EX）のテストケースは「自身のオフセット」に注目したテストです
  • 対応状況（有効なもの）

    凡例： 1 table / 2 table-row / 3 table-row-group / 4 table-column / 5 table-column-group / 6 table-cell / 7 table-caption

    ブラウザ 基点 オフセット
    IE 8/9/10/11 1, 6, 7 1, 6, 7
    Chrome, Safari, Opera 1, 6, 7 1, 6, 7
    Firefox ..9 なし 1
    Firefox 10..29 1 1
    Firefox 30..36 1, 2, 3, 6 1
    Firefox 37.. 1, 2, 3, 6, 7 1, 2, 3, 6

    なお，4 table-column，5 table-column-group は CSS2 および CSS3 仕様上，有効にはならないとされている．
